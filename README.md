# fortnox

[Crystal](https://crystal-lang.org/) shard for working with the
[Fortnox](https://www.fortnox.se/) API.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  fortnox:
    github: kevinsjoberg/fortnox
```
